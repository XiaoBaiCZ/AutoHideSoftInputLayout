# AutoHideSoftInputLayout
自动隐藏键盘布局

基于帧布局实现的自动隐藏键盘布局，使用方法和帧布局一样

- **OnHideListener** 接口监听隐藏事件
- **hideSoftInput** 函数可直接调用隐藏键盘
